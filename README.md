# kbooth
Photobooth Application for printing to ESC/POS Printers.

# Requirements:
cmake
libusb

# External Libraries
- SDL3
- SDLimages
- SDLttf
- Dear ImGui
- libdither
- libusb

# Build

```bash
git clone --recurse-submodules --shallow-submodules https://github.com/thewatermelonman/kbooth.git
```

```bash
 mkdir build && cd build
 cmake ..
 make 
```
